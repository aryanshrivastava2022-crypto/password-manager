# Password Manager
A simple Python Password Manager that helps users store and manage account credentials.

# Features
- Add Account Passwords
- View Saved Accounts
- Search Passwords
- Delete Accounts
- Interactive Menu Interface

# Requirements
- Python 3.x

# How to Run
1. Download the project files.
2. Open the project folder.
3. Open a terminal or command prompt.
4. Run:
password_manager.py
